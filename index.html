About Find-Repo

Welcome to Find-Repo, a GitHub Repositories Finder!

How to Use:

Enter a GitHub username in the input field to discover their repositories. Click the "Fetch Repositories" button to get started.

Clicking on a repository will show additional information such as: Description, Topics, Owner, Repository URL: Visit (Click to navigate to the actual GitHub repository), Created, Last Updated, Language, License, Stars, Forks, Watchers, Open Issues, Open PRs, Size.

Dark Mode:

Toggle dark mode by clicking the moon icon in the top right corner.

Bookmarks:

Bookmark repositories by clicking the bookmark icon. View your bookmarks by clicking the "View Bookmarks" button.

To delete a bookmark, click the trash icon next to it in the "View Bookmarks" modal. You can also navigate to the GitHub repository by clicking its name.
